# Roadmap

Annotator is open source at its heart. The core annotation and conversation experience will always be free and available to everyone.

---

## Current focus

These are things we're actively working on or considering for the near term:

- Improved mobile experience
- Better PDF rendering and annotation positioning
- More file format support
- Keyboard shortcut customization UI
- Annotation search and filtering

---

## Community-driven features

We're interested in building these if there's enough community demand. If any of these would be valuable to you, let us know in [Discussions](https://github.com/donjguido/annotator/discussions).

### Sync & collaboration

- **Cloud sync** — Save and sync your annotations across devices. No more exporting/importing JSON manually.
- **Shared sessions** — Collaborate on the same document with others in real time or asynchronously.
- **Version history** — Track changes to annotations and conversations over time.

### Hosted AI access

- **Subscription for API access** — Use AI models without managing your own API keys. We'd handle the provider integration so you can just pick a model and go.
- **Usage-based pricing** — Pay only for what you use, without needing accounts with individual AI providers.

### Integrations & packaging

- **Obsidian plugin** — Use Annotator's highlight-and-chat workflow directly inside Obsidian vaults.
- **NPM package** — Publish a reusable component for embedding Annotator in other apps (available on request).
- **Dynamic file classification** — Browse and organize your local folders from within Annotator, with interactive detection and handling of document formats.

### Other ideas we're exploring

- Annotation templates and presets
- Batch annotation workflows
- API for programmatic access
- Plugin/extension system

---

## What we won't do

Some things are core to Annotator's identity:

- **Privacy first** — Your documents and API keys stay in your browser. Any sync/cloud features would be opt-in with clear data policies.
- **Open core** — The base tool remains open source and fully functional without paid features.
- **No lock-in** — You can always export everything and take your data with you.
