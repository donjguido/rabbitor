# Keyboard Shortcuts

Annotator ships with default keyboard shortcuts that you can customize.

## Default shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+1` | Switch to **Edit** mode |
| `Ctrl+2` | Switch to **Annotate** mode |
| `Ctrl+Shift+E` | Open **Export** menu |
| `Ctrl+,` | Open **Settings** |

On macOS, use `Cmd` instead of `Ctrl`.

## Customizing shortcuts

Keyboard shortcuts are stored in your browser's `localStorage` under the key `annotator_hotkeys`. Currently, customization is done by modifying this value directly — a UI for rebinding shortcuts may be added in a future release.

## Tips

- In **Annotate mode**, simply select text to create a highlight — no keyboard shortcut needed
- Press **Enter** in the sidebar input to send a message
- Press **Escape** to close the Settings panel or Export menu
